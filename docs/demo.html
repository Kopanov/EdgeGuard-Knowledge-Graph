Relationship Properties
Relationship From → To Properties
USES ThreatActor → Technique sources, confidence_score
ATTRIBUTED_TO Indicator → Malware sources, confidence_score
INDICATES Indicator → Vulnerability sources (Future)
RESOLVES_TO DomainName → IP sources (Future)
📌 Key Implementation Details: All nodes have sources (array of source IDs) and confidence_score (0-1) for data quality tracking. Sector tagging uses word-boundary regex to prevent false positives (e.g., "phishing" ≠ "his" in healthcare).
